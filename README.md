# Job Portal Web Development Project

## Project Title
Job Portal Website

## Description
This project is a simple Job Portal website developed using HTML and CSS.  
The purpose of this project is to demonstrate the basic structure of a job portal where users and companies can register and the admin can log in to the system.

The project contains three main pages:
1. Sign Up Page
2. Company Registration Page
3. Admin Login Page

## Technologies Used
- HTML
- CSS

## Project Structure

jobportal/
│
├── signup.html
├── company.html
├── admin.html
├── style.css
└── README.md

## Features
- User Registration option
- Company Registration form
- Admin Login page
- Simple and clean user interface
- External CSS used for styling

## How to Run the Project
1. Download or copy all the files into a single folder.
2. Open the folder on your computer.
3. Double click on **signup.html** or open it in any web browser.
4. The Job Portal Sign Up page will appear.
